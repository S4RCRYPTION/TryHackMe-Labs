# 🌐 Mission 10: How Websites Work | S4RCRYPTION

Veb-texnologiyalar qanday ishlashini tushunish kiberxavfsizlikning poydevori hisoblanadi. Ushbu laboratoriya ishida veb-saytlarning "sahna orti" jarayonlarini o'rgandim.

### ✅ O'rganilgan asosiy tushunchalar:
* **DNS (Domain Name System)**: Sayt nomlarini IP manzillarga aylantirish jarayoni.
* **HTTP/HTTPS Protocollari**: Client va server o'rtasidagi Request/Response mexanizmi.
* **Web Assets**: HTML, CSS va JavaScript-ning vazifalari.
* **Status Codes**: Serverdan qaytadigan asosiy javob kodlari tahlili.
